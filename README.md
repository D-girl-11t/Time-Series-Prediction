# Time-Series-Prediction

1. Naive approach
2. Simple Average
3. Weighted Moving Average
4. Simple Exponential Smoothing
5. Holts Linear trend 
 - winter method and seasonality

For multivariate,Vector Auto Regression is primary and famous method.
